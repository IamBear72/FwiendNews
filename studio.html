Publishing steps

How this goes live

1. Unlock the newsroom with your private key.

2. Link your site posts.json file once, or import a copy as fallback.

3. Save your article into the newsroom library.

4. After that, Publish can reuse the remembered file in this browser.

5. If the file link is missing, Publish will ask where to save the updated file.

6. Chromium-based browsers usually give the smoothest direct file publishing support.
